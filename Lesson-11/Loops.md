# Loops

Loops are simple repeating statements which are repeated till a specific condition results in false

## Basic loop

while loop Sytax:-

``` javascript

let i=1; //initialization
 
while(i<=5){  //condition
    console.log(i);
    i++; // updation
}
```

for loops

``` javascript

for(let i=1;i<=5;i++){

}
```

> ### Note
>
> We use while for non standard loops and for loop where we need a standard loop.
